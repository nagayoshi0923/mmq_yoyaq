## 使い方（運用ルール）

- **オーナーは番号で指示するだけ**:「DI-17 やって」「DI-05 と DI-06 まとめて」「DI-13 は却下」
- 状態: ⬜未判断 / 👍やる（承認済み・順番待ち） / 🔧対応中 / ✅済み / ❌やらない
- **オーナー追記**: 実機を見て気になった点は、該当ページの下にそのまま1行追記してください（書式自由）。AI が次の整理で DI-番号を振って指示可能な形に変換します。
- 各項目は「コード監査で見つけた候補」であり、**やるかどうかはオーナー判断**。見た目の最終判断は実機優先。
- 大規模なもの（★）はページ改装バッチ（ADMIN_UI_REDESIGN_PLAN）として実施。小粒はそれ単体で1コミット。
- 🔒 公演カード・PerformanceModal は対象外（見た目変更禁止）。顧客向けページも対象外（ブランド維持・トークン化のみ）。
