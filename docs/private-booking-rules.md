# 貸切公演 時間調整ルール

## 基本設定

| 項目 | 値 | 備考 |
|------|-----|------|
| 最早開始時間 | 9:00 | これより早い開始は不可 |
| 最遅終了時間 | 23:00 | これより遅い終了は不可 |
| 公演間の準備時間 | 60分 | 清掃・準備・休憩 |

---

## 公演枠の基本時間

| 枠 | デフォルト開始 | 備考 |
|----|--------------|------|
| 朝公演 | 10:00 | シナリオ時間により9:00まで早める可能性あり |
| 昼公演 | 14:00 | 朝公演終了+1時間後 |
| 夜公演 | 18:00 | 昼公演終了+1時間後 |

---

## 時間調整ルール

### 朝公演の調整

| 条件 | 対応 |
|------|------|
| 朝公演が13:00以降に終わる | 朝公演開始を早める（最早9:00） |
| 朝公演を9:00にしても13:00に終わらない | 昼公演開始を遅らせる |

### 昼公演の調整

| 条件 | 対応 |
|------|------|
| 昼公演が17:00以降に終わる | 夜公演開始を遅らせる |
| 昼公演終了+1時間後が18:00を超える | 夜公演開始を昼公演終了+1時間に調整 |

### 夜公演の調整

| 条件 | 対応 |
|------|------|
| 夜公演が23:00を超える | 夜公演開始を早める |
| 開始を早めても23:00を超える | 夜公演は不可（1日2公演のみ） |

---

## シナリオ時間別 公演スケジュール例

準備時間60分を考慮した場合の例：

| シナリオ時間 | 朝公演 | 昼公演 | 夜公演 | 備考 |
|-------------|--------|--------|--------|------|
| 120分（2h） | 10:00〜12:00 | 14:00〜16:00 | 18:00〜20:00 | 標準 |
| 150分（2.5h） | 10:00〜12:30 | 14:00〜16:30 | 18:00〜20:30 | 標準 |
| 180分（3h） | 10:00〜13:00 | 14:00〜17:00 | 18:00〜21:00 | 標準 |
| 210分（3.5h） | 9:30〜13:00 | 14:00〜17:30 | 18:30〜22:00 | 朝公演を30分早める |
| 240分（4h） | 9:00〜13:00 | 14:00〜18:00 | 19:00〜23:00 | 朝公演を1時間早める、夜公演を1時間遅らせる |
| 270分（4.5h） | 9:00〜13:30 | 14:30〜19:00 | - | **1日2公演のみ** |
| 300分（5h） | 9:00〜14:00 | 15:00〜20:00 | - | **1日2公演のみ** |
| 360分（6h） | 9:00〜15:00 | 16:00〜22:00 | - | **1日2公演のみ** |

## 1日の最大公演数

| シナリオ時間 | 最大公演数 |
|-------------|-----------|
| 〜240分（4h） | 3公演 |
| 241分〜300分 | 2公演 |
| 301分〜 | 2公演（または1公演） |

## 注意事項

1. **お客様への案内**
   - 貸切リクエスト時「開始時間は店舗の状況により調整させていただく場合があります」と明記
   - 時間変更が必要な場合は、お客様に確認してから確定

2. **店舗側の運用**
   - 上記ルールを参考に、実際のスケジュールを調整
   - 長時間シナリオは事前に時間調整の可能性を伝える

3. **更新履歴**
   - 2026-01-02: 初版作成
