# ⚠️ アーカイブ（参照専用）

> **警告**: このフォルダ内のドキュメントは**古い可能性があります**。
> 現在のシステムと一致している保証はありません。

---

## このフォルダの性質

| 項目 | 説明 |
|------|------|
| **信頼性** | 低 - 現在のコードと一致している保証なし |
| **更新義務** | なし - 参照専用として保持 |
| **削除可否** | 可 - 明らかに不要なものは削除してよい |

---

## 含まれるドキュメント

### phase-reports/
過去のフェーズ完了報告。実装は完了しているが、記載内容は当時のもの。

### refactoring/
過去のリファクタリング計画。実装済みだが、現在のコード構造と異なる場合がある。

### aws-ses/
旧AWS SES設定ドキュメント。**Resendに移行済み**のため参照不要。

### misc/
その他の古いドキュメント（パフォーマンス改善、TODO、開発状況等）。

---

## 新しい情報が必要な場合

アーカイブではなく、以下の最新ドキュメントを参照してください：

- `docs/index.md` - ドキュメント目次
- `docs/pages.md` - ページ一覧
- `docs/development/` - 開発ガイドライン（最新）
- `docs/setup/` - セットアップ手順（最新）

---

## ドキュメントをアーカイブから戻す場合

1. 内容が現在のコードと一致しているか確認
2. 最終確認日を追記
3. 適切なフォルダに移動
4. `docs/index.md` を更新
