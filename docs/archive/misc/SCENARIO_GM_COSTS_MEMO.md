# シナリオGM給料メモ

## gm_costs フィールドの構造

シナリオの `gm_costs` 配列には、同じロール（main）で2つの金額が設定されている場合がある。

### 2つの金額の意味
| 順番 | 意味 | 備考 |
|------|------|------|
| 1つ目 | **正規料金** | 通常公演時のGM報酬 |
| 2つ目 | **GMテスト料金** | GMテスト公演時のGM報酬（正規より低い） |

### 例
```json
{
  "title": "立方館",
  "gm_costs": [
    { "role": "main", "reward": 10750 },  // 正規料金: 10,750円
    { "role": "main", "reward": 8250 }    // GMテスト料金: 8,250円
  ]
}
```

### 給料レンジの傾向
| 給料レンジ | 代表シナリオ |
|------------|--------------|
| 10,000円以上 | 立方館（10,750円/8,250円）|
| 9,000円台 | 流年、霧に眠るは幾つの罪（9,750円/7,250円）|
| 8,000円台 | BBA、告別詩、探偵撲滅（8,750円/6,250円）|
| 7,000〜8,000円 | 多数のシナリオ |
| 5,000〜6,000円 | 中堅シナリオ |
| 2,000〜4,000円 | 短時間シナリオ（OVER KILL、ゼロ・オルビット等）|
