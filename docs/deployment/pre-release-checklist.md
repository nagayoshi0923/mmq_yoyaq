## 📋 チェックリストの使い方

このチェックリストは、本番環境へのリリース前に**必ず全ての項目を確認**してください。
各項目を確認したら `[ ]` を `[x]` にチェックしてください。

**重要**: チェックリストの項目は優先度順に並んでいます。🔴 高優先度項目は絶対にスキップしないでください。
